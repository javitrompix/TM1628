
Libreria para el uso del frontal completo de un DVD que tenga el control basado en el TM1628, SM1628 o el MC2101.
Estos integrados se conocen como "DVD LED driver".

La placa entera de frontal de DVD suele tener seis cables.

STB, CLK y DATA	 Para la comunicación con el Driver	
GND y +5V        Para la alimentación de la tarjeta
IR		 Para la trama del mando de infrarojos

Normalmente suele venir serigrafiado en la placa el pineado.
